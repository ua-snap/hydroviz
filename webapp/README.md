# hydroviz-webapp

## Setup

```
nvm use lts/jod
npm install
```

## Running for development

Run the [local data API](https://github.com/ua-snap/data-api/) from the `hydroviz_prototype_api` branch.

Running the webapp:

```
export SNAP_API_URL=http://localhost:5000
cd hydroviz-webapp
npm run dev
```
